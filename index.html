A Rede de Geração Cidadã de Dados nasceu da união de organizações marginalizadas para democratizar a pauta de dados e censos populares no Brasil. Criada a partir do I Seminário de Geração Cidadã de Dados em setembro de 2024, na cidade do Rio de Janeiro, a Rede propõe metodologias para que a sociedade civil produza, analise e distribua dados de interesse público, valorizando conhecimentos territoriais e populares.

A Geração Cidadã de Dados (GCD) surge num contexto no qual a produção centralizada de dados limita a eficácia das políticas públicas e distancia a população das tomadas de decisão que impactam as suas vidas. A Rede GCD se apresenta como uma alternativa a esse modelo, oferecendo meios e tecnologias para a geração de dados por e para a comunidade. Ao estimular a produção de dados para territórios populares e populações marginalizadas, de forma a contribuir para uma cidade mais justa e feliz.

Os meios, as temáticas e os atores que aplicam a GCD são diversos, porém defendemos alguns pressupostos em comum para a Geração Cidadã de Dados:

  • priorizar o protagonismo de grupos marginalizados e sub-representados, com ênfase em dados que valorizem suas histórias;
  • defender o uso de softwares livres, tecnologias acessíveis e bases de dados abertas;
  • tornar o dado acessível por meio de leitura, escrita e outras formas de linguagem que ultrapassem as fronteiras populares;
  • contribuir para a formação de agentes sociais por meio da troca de saberes.

A GCD é uma ferramenta para identificar demandas, monitorar, comparar e complementar a produção de dados oficiais, fornecendo evidências para a tomada de decisões. Acreditamos que a Geração Cidadã de Dados é uma tecnologia para a geração pública, de interesse público e por autoridades responsáveis, que podem ter na GCD uma tecnologia útil para a gestão pública.

Acreditamos que isso será possível somente a partir da articulação coletiva, então junte-se a nós para democratizar e descentralizar o dado e contribuir para uma produção de conhecimento mais democrática e representativa.
